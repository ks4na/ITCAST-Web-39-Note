# 数组对象

## 创建数组对象

```js
// 1. 数组字面量 - []
var arr = [1, 2, 3]

// 2. 数组构造函数 
var arr = new Array(1, 2, 3)
console.log(arr)  // [1,2,3]
// 注意： var arr = new Array(4)创建的是长度为4且每项为undefined的空数组
```

## 判断一个变量是否为数组对象

```js
// 1. instanceof
var arr = []
console.log(arr instanceof Array)  // true
// 2. Array.isArray()
var arr = []
console.log(Array.isArray(arr))  // true

// 两者区别：instanceof假定是**单一的全局执行环境**，而如果有多个frame则必须使用Array.isArray()
```

### 带参数的sort()如何实现？

> 以之前的冒泡排序为例进行模拟。

```js
function bubbleSort(arr, compareFn){
    // 先定义一个默认的比较函数
    var _compare = function (a, b){
        return a - b
    }
    if(compareFn && typeof compareFn === 'function'){
        _compare = compareFn
    }
    for(var i = 0; i < arr.length-1; i++){
        var isSorted = true  // 假设已经排好序了
        for(var j = 0; j < arr.length-1-i; j++){
            if(_compare(arr[j], arr[j+1]) > 0){
                var temp = arr[j]
                arr[j] = arr[j+1]
                arr[j+1] = temp
                
                isSorted = false // 如果发生交换则代表还没有排好序
            }
        }
        if(isSorted){
            break
        }
    }
}
```

### 数组去重

```js
// 1. 使用indexOf()方法
function unique(arr){
    var filteredArr = []
    for(var i = 0; i < arr.length; i++){
        if(filteredArr.indexOf(arr[i]) === -1){
            filteredArr.push(arr[i])
        }
    }
    return filteredArr
}

// 2. 【推荐】使用对象键名唯一的特性 - 此方法可以同时统计每一个元素出现的次数
function unique(arr){
    var obj = []
    for(var i = 0; i < arr.length; i++){
        if(obj[arr[i]]){
            obj[arr[i]] ++
        }else{
            obj[arr[i]] = 1
        }
    }
    console.log(obj)  // 打印数组中每一个元素及其对应的出现次数
    var filteredArr = []
    for(var key in obj){
        filteredArr.push(key)
    }
    return filteredArr
}
```
